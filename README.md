## ðŸ› ï¸ How to Deploy Your Own Version
Follow these steps to make this website your own:

1.  **Fork this Repository:** Click the "Fork" button at the top right of this page.
2.  **Edit the Code:**
    * Open `index.html`.
    * Change the **Name**, **Links**, and **Projects** to match your own profile.
3.  **Commit Changes:** Save your changes and commit them to your repository.
4.  **Activate GitHub Pages:**
    * Go to **Settings** > **Pages**.
    * Under **Source**, select `Deploy from a branch`.
    * Select `main` branch and `/ (root)` folder.
    * Click **Save**.
5.  **Done!** Refresh the page after 1-2 minutes to see your generated link.
